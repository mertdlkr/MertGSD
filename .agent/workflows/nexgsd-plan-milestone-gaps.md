# NexGsd Plan Milestone Gaps

Convert milestone gaps into actionable follow-up work.

## Steps

1. Read the latest milestone audit report.
2. Separate:
   - critical blockers
   - missing requirements
   - integration issues
   - non-critical tech debt
3. Decide whether each item needs:
   - inserted decimal phase
   - new normal phase
   - direct addition to an upcoming phase
4. Update `ROADMAP.md` and `STATE.md` accordingly.
5. If needed, create a first follow-up plan prompt for the next gap-closing phase.

## Completion

Recommend the exact next planning command for the chosen gap phase.
