# NexGsd Remove Phase

Remove only unstarted future phases.

## Steps

1. Read `ROADMAP.md` and `STATE.md`.
2. Confirm the target phase is not already executed or partially executed.
3. Remove the phase entry from roadmap documents.
4. If the phase has a directory with no completed work, remove or archive it.
5. Renumber later phases only if that keeps the roadmap simpler and safer.

## Completion

Report what was removed and any renumbering performed.
