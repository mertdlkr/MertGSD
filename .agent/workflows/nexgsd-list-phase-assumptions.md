# NexGsd List Phase Assumptions

Preview planning assumptions before writing plans.

## Steps

1. Read `ROADMAP.md`, `REQUIREMENTS.md`, `STATE.md`, and the target phase section.
2. Read phase `CONTEXT.md` and `RESEARCH.md` if they exist.
3. Output a concise list of:
   - likely implementation direction
   - dependencies you expect
   - likely files or layers involved
   - open questions or risks
4. Do not create or modify plan files in this workflow.

## Output

Return assumptions only. If the user approves, point them to `/nexgsd-plan [N]`.
